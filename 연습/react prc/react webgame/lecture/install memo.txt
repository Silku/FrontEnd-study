
[1] 순서
==================
원하는 폴더에 터미널로 진입 ex) lecture
1. package.json 생성
    - npm init

2. react, react-dom 생성
    - npm i react react-dom

3. webpack 생성
    - npm i -D webpack webpack-cli

********* 
    -D 는 개발용으로 쓰인다는것을 의미함.
    package.json에도 아래처럼 devDependencies라고 따로 등록됨.
  "devDependencies": {
    "webpack": "^5.74.0",
    "webpack-cli": "^4.10.0"
  }
*********
